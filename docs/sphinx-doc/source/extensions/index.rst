#############################
Extensions
#############################

TI's OpenCL implementation has been extended with a set of features beyond the
OpenCL 1.1 specification. These features were added in order to better support
the execution of code on the C66 DSP, to enable existing DSP libraries, and to
better map to TI's devices.
